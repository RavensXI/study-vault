# NotebookLM Video Overviews - Task List

GCSE revision lessons that need a NotebookLM podcast-style video overview.

## How to create each video

1. Open the lesson URL below (log in first with the demo account)
2. Copy ALL the lesson text content (everything inside the main article)
3. Go to https://notebooklm.google.com/
4. Create a new notebook
5. Paste the lesson text as a source
6. Generate an Audio Overview
7. Once generated, save/download the audio overview
8. Note down the lesson identifier and the resulting video/audio URL

- Site: https://study-vault-alpha.vercel.app/
